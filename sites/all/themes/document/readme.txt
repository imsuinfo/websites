At the moment this readme is just a collection of notes.


The different css media types are treated as follows:

- View Only: embossed, print, tv
- View, Unhidden Edit: aural, braille, tty
- View, Hidden Edit: handheld, projection, screen

- screen
  - View, Edit with hidden menus (unhides on hover/active)
  - child media types:
    - handheld, projection

- print
  - View only, no edit controls
  - child media types:
    - embossed, tv

- braille
  - View, Edit with visible menus
  - child media types:
    - aural, tty
